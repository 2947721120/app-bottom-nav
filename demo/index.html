A bottom navigation element, designed to comply with the Material Design specifications.

The app-bottom-nav element should only have between 3 and 5 app-bottom-nav-item elements. The value of "selected" should match one of the labels.

NOTE: By default, this demo will only be visible when the viewport is narrower than 640px. This is the default app-drawer-layout responsive width and means the app-bottom-nav will show when the app-drawer retracts. To overcome this function, the paper-buttons below can override the visible state of the app-bottom-nav.
